# Local Guide Translator - Product Definition

## Product Overview
Local Guide Translator is an AI-powered tool that translates regional slang and provides local insights using context-aware language models. It helps users understand local nuances, street food recommendations, and traffic patterns in specific regions.

## Regional Context
- **Focus Areas**: Bengali, Hindi, Tamil, Marathi speaking regions
- **Use Cases**: 
  - Slang translation for street food names
  - Local recommendations (restaurants, places)
  - Traffic and commute estimation

## Key Features
1. **Slang Translator**: Converts regional slang to standard Hindi/English
2. **Street Food Recommender**: Suggests local food based on cuisine preferences
3. **Local Traffic Estimator**: Estimates commute time based on local knowledge

## Implementation
Built with Python using OpenAI API and custom context files for regional data.
